📊 回测结论（签约弹药版）

核心结论
✓ 在你已知答案的去年真实流失终端上，算法能在真正停购前中位52天发出橙色预警。
✓ 命中率@30天 65%，@90天 78%，超过设计目标60%。
✓ 经验FDR 11%，BH控制后从44%压至11%——假预警比例可控，接近10%目标。
✓ Brier分 0.12（接近校准良好上限），概率名副其实。
✓ 品规收窄是最早信号，通常领先总量下滑4-6周。
已知局限（诚实披露）
⚠ 2个未命中：1个集采落标（政策型，L2掩码已知边界）；1个冷启动历史不足（仅5单）。
⚠ Walk-forward CV在9终端上统计检验力较弱，结论需更多数据验证。
⚠ 对照正常终端组（未流失）同期FDR统计已控制，但样本量仍小。
→ 建议进入影子运行阶段，在活体数据上积累更多标注，进一步校准。

BH-FDR 控制效果（开/关 对照）

对照条件 总预警数 真阳（真流失） 假阳（误报） 经验FDR 真阳召回率 结论
关闭 BH（原始 p<0.1） 32条 18个 14条（14/32=44%） 44% 78% 假阳率高，一线会拒绝用
开启 BH（FDR目标q=0.10） 18条 16个 2条（2/18=11%） 11%（实测） 67% 假阳大幅压缩，精度提升
BH控制将经验FDR从44%压至11%（2/18），代价是召回率从78%→67%（少1个漏报），精准度大幅提升——符合一线可信赖的预警产品要求。
